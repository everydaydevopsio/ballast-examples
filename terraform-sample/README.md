# Terraform Sample

This directory is a test case for [Ballast](https://github.com/everydaydevopsio/ballast)'s Terraform/IaC language detection and rule installation.

## Purpose

This sample demonstrates how Ballast automatically detects a Terraform project and installs appropriate agent rules for Infrastructure as Code (IaC) development.

## What This Tests

- Automatic Terraform language detection (via `.tf` files or Terraform module structure)
- Installation of Terraform-specific linting and best-practice rules
- Rule file generation for Terraform projects

## Running Ballast on This Sample

From this directory:

```bash
ballast install --target cursor --agent linting --yes
```

This will generate `.cursor/rules/terraform-linting.mdc` with Terraform-specific linting guidance.

## Contents

- `.ballast/` — Ballast configuration directory
- `.cursor/` — Cursor/Codeium AI editor integration (generated by Ballast)
- `.rulesrc.json` — Rules configuration
